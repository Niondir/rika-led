./software/RIKA-LED.sln
=========================
Beinhaltet die 3 Windowsanwendungen. Diese können mit Visual Studio 2008 kompiliert werden.

Ein API Dokumentation ist in Form von XML Kommentaren im Quellcode sowie als HTML Dokumentation in ./Doku.rar/Index.html


Schild und Lampe
================
Können mit AVR Studio compiliert und auf einen AT Mega 8 geschrieben werden.
Detailierte Informationen sind im Ordner ./software/Lampe/doku/ zu finden.

Sowie für die Schilder in ./software/Schilder/doc/ und ./software/Schilder/Installation.txt
